Reporting

  • Learning data

    The data (management information) provided in reports is based on the profile details supplied by learners when they registered.

    Profile details can be updated by learners at any time.

    The data is provided as CSV files (a data format that is easily exported to other programs).

  • Registered learners

    Provides listings of learners registered to departments.

  • Booking information

    Shows data on classroom courses, identifying requests (state before payment accepted), bookings (state once payment received) and cancellations (for reasons such as capacity issues, course not run or payment not accepted).

  • Learners by profession

    Displays how many learners have selected a specific profession as their primary area of work.
